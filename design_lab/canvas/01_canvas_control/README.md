# 01 Canvas Control

This study isolates the bottom-center canvas control dock.

Scope:

- dock shape and spacing
- icon design for pointer / hand / undo / redo
- zoom pill styling
- right-side zoom popout menu
- relationship to the dotted dark canvas background

Reference goals:

- match the compact, dark, soft-cornered control cluster
- make the active hand tool read clearly
- tune icon weight so it feels precise instead of placeholder-like
- keep the zoom menu visually attached to the percentage control

Current choices:

- static canvas stage with dotted grid
- inline SVG icons instead of text glyphs
- click-to-toggle zoom menu so the panel can be reviewed in context

Things to judge:

- do the icon shapes feel close enough to the reference?
- is the dock too tall or too wide?
- should the active tool use this exact pale-yellow tone?
- should the zoom popout sit tighter to the percentage pill?
