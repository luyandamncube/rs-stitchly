# Data Sources Lab

This folder is the isolated review space for Stitchly data-catalog and source/sink browser studies.

Purpose:

- design the canvas popup for workspace data browsing
- review how local DuckDB catalogs should appear before production wiring
- test how source and sink selection should connect back to nodes

What belongs here:

- canvas popup data-source windows
- DuckDB database, schema, and table browser studies
- source-picker and sink-picker interaction concepts
- table preview, binding, and catalog metadata experiments

Current convention:

- `v1/00_xxx/`
  Ordered data-source popup studies for the first pass.

Current versions:

- `v1/`
  First pass for the canvas-based data-sources popup.
