# 03 Table Overview

Purpose:

- build on the schema-grain explorer from `02_catalog_overview`
- move the explorer one level deeper into a selected table
- use the Overview tab to browse table columns, types, and descriptions

Included in this study:

- the same left-side catalog tree shell
- breadcrumb extended down to the selected table
- table-grain title row in the explorer
- `Overview` and `Sample Data` tabs, with `Overview` active
- column browser with filter, count, and `Column / Type / Description` fields

What to review:

- whether this table-grain step feels like a natural continuation from the schema view
- whether the table Overview should stay this simple before adding sample-data and details tabs later
- whether the column list is dense enough without becoming hard to scan
- whether descriptions deserve more width than types in this layout
