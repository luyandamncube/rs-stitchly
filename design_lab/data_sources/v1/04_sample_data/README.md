# 04 Sample Data

Purpose:

- build on the table-grain explorer from `03_table_overview`
- switch the active tab from `Overview` to `Sample Data`
- test how a Databricks-like sample grid should sit inside Stitchly's darker popup language

Included in this study:

- the same left-side catalog tree shell
- breadcrumb extended down to the selected table
- `Overview` and `Sample Data` tabs, with `Sample Data` active
- sample preview header with utility actions
- sample data grid with row numbers and typed columns

What to review:

- whether the sample grid feels dense but still readable inside the popup
- whether the top-right utility icons are enough before adding richer query controls
- whether this grid should stay plain or inherit more Stitchly accent styling
- whether the current column widths are balanced for mixed identifiers and metrics
