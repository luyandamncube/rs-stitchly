# Integrations Lab

This folder is the isolated review space for Stitchly connector and integration-management studies.

Purpose:

- design the canvas popup for integrations and external accounts
- review connector-list density and create-menu behavior before production wiring
- keep integration management visually aligned with the existing canvas popup language

What belongs here:

- canvas popup integration windows
- connector list / table studies
- create-connector menu treatments
- connector row density, icon, and metadata experiments

Current convention:

- `v1/00_xxx/`
  Ordered integrations popup studies for the first pass.

Current versions:

- `v1/`
  First pass for the canvas-based integrations popup.
