# 00 Integrations Window

Purpose:

- explore a first canvas-popup integrations window
- carry over the `Runs & Logs` popup shell while simplifying the inside dramatically
- test whether a table with service mark, name, created date, and comment is enough for first-pass connector management

Included in this study:

- left menu rail with an active `Integrations` item under `Runs`
- popup window anchored to the rail
- `Integrations` header with a compact `+` action
- visible connector-create dropdown tray outside the panel with the first connector set
- simplified single-line integrations table with service symbol, name, created-at, and comment columns

What to review:

- whether the simplified panel feels consistent with the existing canvas popups
- whether the outside `+` dropdown keeps the panel compact enough
- whether the tighter one-line table row density feels right for connection management
- whether the service marks are clear enough before we later swap to production icons
