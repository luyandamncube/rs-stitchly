# Login Lab `v1`

This folder contains the first isolated design studies for Stitchly login and authentication surfaces.

Purpose:

- analyze reference login screens before adapting them
- refine shell, form, and brand-panel patterns in isolation
- approve visual language before production implementation

Convention:

- `shared.css`
  Shared login-language stylesheet for the current auth design pass.
- `00_xxx/`
  Reference analysis and foundational direction.
- `01_xxx/`
  First visual shell studies.
- `02_xxx/`
  Form and component refinement studies.

Preferred progression:

1. reference analysis
2. shell layout study
3. form study
4. brand-panel study
5. mobile adaptation

This folder is static-first and should remain separate from the production app shell.
