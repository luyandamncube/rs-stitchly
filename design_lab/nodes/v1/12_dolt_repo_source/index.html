Why this node stops here

Export format, changed-table selection and raw artifact retention belong on dolt_dump or a later dolt_diff_export node, not on the repo source itself.

Checkpoint visibility

The source card shows the current synced commit, but full checkpoint ownership can stay in a dedicated checkpoint node or hidden runtime table until we need it exposed.

Execution timing

Best downstream pair `dolt_dump` for bootstrap, `dolt_change_manifest` for resync
