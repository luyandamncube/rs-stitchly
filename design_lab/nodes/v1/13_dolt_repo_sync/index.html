Boundary

This node should not decide which tables changed or how they get exported. It only advances the repo and resolves the commit range.

Checkpoint ownership

Previous commit state is workflow-managed and should not be manually typed into this node. The sync step only turns that checkpoint into a usable runtime range.

Bootstrap path

When no checkpoint exists yet, the initial load should stay on the bootstrap path with dolt_dump rather than pretending a recurring diff window already exists.
