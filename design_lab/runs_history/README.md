# Runs History Lab

This folder is the isolated review space for Stitchly run-history window studies.

Purpose:

- design the canvas popup for run history and logs
- review run-list density, filters, and metrics before production wiring
- keep run-history browsing separate from node-state and workflow-run-state studies

What belongs here:

- canvas popup run-history windows
- runs list / logs list / timeline list surfaces
- filter, search, export, and summary metric treatments
- menu-item-triggered run browsing concepts

Current convention:

- `v1/00_xxx/`
  Ordered run-history window studies for the first pass.

Current versions:

- `v1/`
  First pass for the canvas-based run-history popup.
