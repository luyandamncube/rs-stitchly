Pending Direction

The node is already part of the current run, but it is still blocked by graph order or upstream completion. It should feel quieter than `ready`.

Muted shell Keep the full node visible, but reduce energy and contrast.
Handle de-emphasis Pending handles should not feel active or schedulable yet.
No success or failure semantics It should read as "not yet", not "done" or "problem".
