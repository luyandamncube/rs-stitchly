## Status Meanings

- `raw_idea`
  Captured but not normalized.
- `scoring`
  In the middle of evaluation.
- `validate_first`
  Worth testing, but not ready for full build commitment.
- `build_now`
  Strong enough to enter active build planning.
- `incubating`
  Interesting, but blocked by weaker economics, unclear data, or lower urgency.
- `live`
  Already launched and operating.
- `archived`
  Intentionally rejected or retired.

## Portfolio Rule

This table should always reflect the latest scored state, not a historical log of every past opinion.

If an idea changes materially, update:

- weighted score
- recommendation
- status
- next action
- last reviewed
